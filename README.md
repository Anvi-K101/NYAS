# GridSync AI

GridSync AI is a research-driven project exploring how predictive artificial intelligence can stabilise grid frequency in renewable-heavy energy systems.

## Focus Areas

- Solar grid instability
- Frequency collapse prevention
- Predictive load balancing
- Coordinated inverter response

## Highlights

- High-accuracy predictive modelling
- Increased safe renewable penetration thresholds
- Cost-efficient grid optimisation strategies

## Format

This project is presented as a structured, static research portfolio rather than a traditional application.

## Technology

- Static site technologies
- Netlify hosting
